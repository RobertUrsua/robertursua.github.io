Github

For LA Hacks, our team created a platform that helps the hungry locate free and readily available sustenence. Our idea was to coordinate with local food banks, participating restaurants and welfare organizations so we can provide updated information on where, and how much food is available. Our hack was awarded "Best Civic Hack" by the City of LA

My role in this project was implementing the map interface used to locate participating food places and to direct users to their selected location. I also designed the login, signup and map pages to fit the selected color palette and attempted to clean up the CSS in the little time that we had. The most challenging technical issue that I addressed succesfully was integrating our individually made HTML, CSS and Javascript files into the Angular framework that my team mate created.

Technical Details: I used Google's Map and Places API through Javascript for the food places map.

  • Date: Apr 2017
  • Technologies: Javascript, HTML/CSS, Node.js, Angular, Firebase, Google APIs

Bruin Nav

Github - Project Specification

For my Data Structures and Algorithms course, I created a text based turn-by-turn navigation system that provides optimal routes for trips within the city of Los Angeles. I implemented the A* algorithm and created optimal data structures in order to ensure quick query processing. Music Visualizer

Circuit board

A section of the schematic

Previous Next

Github - Schematic - Video of Operation

I created an Arduino operated device that takes input via 3.5mm audio jack and creates a light show using an LED matrix. I designed and soldered the circuit board, and I wrote the Arduino code for multiplexing the LED Matrix.

The devices uses an MSGEQ7 IC to perform spectrum analysis on the audio and adjusts the light show based on the audio's frequency components. It also uses four 74HC595 shift registers to facilitate multiplexing.

  • Date: Nov 2016
  • Technologies: Arduino, C/C++, Cadsoft EAGLE
